# JSSM4

## 一个 SM4 加解密 JavaScript 实现

解决了 js 中循环左移位数丢失问题；

```
密钥：qawsedrftgyhujik

原文： abc
index.html:20 加密耗时: 0.198974609375ms
index.html:21 密文： MYj7/IK7Sc1lbZQewXNVdg==
index.html:24 解密耗时: 0.175048828125ms
index.html:25 解密： abc
```
